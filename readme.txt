Displays the cost of shipping the product beneath the add to cart button. The shipping cost updates when the quantity changes.

== Description ==

This plugin displays the shipping cost under the add to cart button.

The price that is displayed is the increase in shipping costs if the product is added to the cart.

This is a basic plugin without any options. If you would like an advanced plugin that adds a shipping calculator to your products, WooCommerce Shipping Calculator on Product Page is the plugin you are looking for https://codecanyon.net/item/woocommerce-shipping-calculator-on-product-page/11496815?ref=wpshowcase.

== Installation ==

1. Install and activate the plugin through the 'Plugins' menu in WordPress.
2. The shipping costs now display below the add to cart button on the product pages (no need to edit any settings).
